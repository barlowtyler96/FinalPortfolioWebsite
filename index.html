CulinaryShares Angular
Technologies used:
Bootstrap 5, Azure AD B2C, RxJS, OAuth 2.0, OIDC

This project showcases Azure Active Directory B2C + Angular. The app allows people from around the world to share their favorite recipes. Users can create an account, sign in, edit their profile, share recipes, and explore other users' shared recipes. User's can also favorite recipes and view recipes that they have shared all in one place. All recipe data is posted and retrieved by my ASP.NET Core Web API.

CulinaryShares Web API
Technologies / Libraries used:
Azure AD, OAuth 2.0, SQL Server, Dapper, Postman, XUnit w/ Fluent Assertions
Deployed on Azure Web Apps
This solution consists of two projects, a class library for data access and a web API. My data access library utilizes dapper with stored procedures. The library is referenced in my API. The API is then consumed by my Blazor/Angular app. My favorite feature is the server-side paging. When a user searches for a recipe, only the results for the current page are returned and stored in memory.
