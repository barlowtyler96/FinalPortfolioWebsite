MY WORK
Recipes Blazor Server

Technologies used:

Bootstrap 5, Azure AD B2C, OAuth 2.0, OpenID Connect, Razor Components


This is a Blazor Server application that I made to consume the Recipes Web API. This app calls the Recipes Web API after obtaining an access token using the client credentials flow and a client secret. The app also uses Azure AD B2C to allow users to create an account and sign in. Once a user is authenticated, they are authorized to view a page where they can submit recipes for review. I made a short video showing off the project in the repositories ReadMe.

Recipes Web API

Technologies used:

Azure AD, OAuth 2.0, SQL Server, Dapper, Postman


This solution consists of two projects. A data access class library that is referenced in my Web Api project. The class library uses Dapper and stored procedures with SQL Server. The API is secured using Azure AD. An access token is granted for a Blazor server app using the client credentials flow (see the Recipes Blazor Server app).
